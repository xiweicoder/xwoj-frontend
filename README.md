# xwoj-frontend

## 项目安装和部署

### 1、安装项目依赖

```bash
npm install
# 
yarn 
```

本项目使用到的依赖/插件（不需执行）

```bash
# 组件库
yarn add --dev @arco-design/web-vue 
# Axios 工具
yarn add axios
# MarkDown 编辑器依赖
yarn add @bytemd/vue-next

# 代码编辑器
yarn add monaco-editor
# 将webpack和文本编辑器整合在一起，便于打包和安装
yarn add monaco-editor-webpack-plugin 
```

### 2、项目构建

```bash
yarn build
```

### 3、生产环境运行

```
yarn serve
```

### 修复文件

```
yarn lint
```
